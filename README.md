# XR Interaction Toolkit Examples Project (For Learning)

A hands-on exploration of Unity's [XR Interaction Toolkit (XRI)](https://docs.unity3d.com/Packages/com.unity.xr.interaction.toolkit@3.0/manual/index.html) — used for **learning and experimentation** with XR interactions in Unity.

## Project Overview

The project demonstrates various XR interaction features organized as stations in a ring-shaped scene:

| Station | Feature | Description |
|---------|---------|-------------|
| 1 | **Locomotion Setup** | Built-in locomotion options and configuration |
| 2 | **Grab Interactables** | Basic object manipulation |
| 3 | **Activate Interactables** | Objects triggered by user actions |
| 4 | **Socket Interactors** | Objects snapping to specific positions |
| 5 | **Gaze Interaction** | Eye-tracked or head-based gaze interaction |
| 6 | **Focus Interaction** | Interaction with focused objects |
| 7 | **2D UI** | World-space 2D UI creation and interaction |
| 8 | **3D UI** | 3D constrained controls |
| 9 | **Physics Interactables** | Combining physics with XR input |
| 10 | **Climb Interactables** | Climbing interaction |

## Getting Started

### Requirements
- **Unity 2021.3** or later
- **Windows Build Support (IL2CPP)** module for PC builds
- **Android Build Support** for Meta Quest builds

### Setup
1. Clone this repository:
   ```bash
   git clone https://github.com/behnoudshafizadeh/XR-Interaction-Toolkit-Examples_Project-for-learning-.git
   ```
2. Open **Unity Hub** → **Projects** → **Add** → Select the `XR-Interaction-Toolkit-Examples-main` folder
3. Open the main scene at:
   ```
   Assets/XRI_Examples/Scenes/XRI_Examples_Main
   ```
